## 三、各阶段开发内容

### 阶段一：Vue 3 SPA（[ ] 进行中 / [x] 已完成）

- [ ] 使用 Vue 3 Composition API 实现随机推荐、分类筛选、菜品详情等核心页面
- [ ] 基于 TypeScript 定义完整的菜品、用户、食材等业务类型，确保前后端类型安全
- [ ] 封装可复用的 Composable 函数，抽离至 core 包供后续多端复用
- [ ] 使用 Pinia 管理用户偏好状态，Vue Router 实现路由守卫与页面级权限控制

### 阶段二：Express 后端（[ ] 进行中 / [x] 已完成）

- [ ] 使用 Express + TypeScript 构建 RESTful API 服务，独立完成路由设计、中间件开发
- [ ] 基于 Prisma ORM 设计数据库模型，管理用户、菜品、分类、收藏等多表关联关系
- [ ] 实现 JWT 用户认证系统，包含注册、登录、接口鉴权、角色隔离等完整流程
- [ ] 开发个性化推荐引擎，基于用户口味偏好、忌口、卡路里目标等画像进行加权推荐

### 阶段三：Nuxt 3 + uni-app（[ ] 进行中 / [x] 已完成）

- [ ] 使用 Nuxt 3 重构 Web 端，实现 SSR 服务端渲染，解决 SEO 与首屏加载问题
- [ ] 基于 uni-app 开发小程序端，复用 core 层核心逻辑，实现一套代码多端覆盖
- [ ] 搭建管理后台，实现菜品管理、用户管理、数据统计看板等完整功能模块
- [ ] 项目部署至 Vercel + 云数据库，配置自动化 CI/CD，实现代码推送即自动上线
