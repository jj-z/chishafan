# 吃啥饭 - 项目需求文档

> 一个解决“今天吃什么”的全栈多端应用。
> 网页、小程序、App 全覆盖，独立完成前后端与部署。

---

## 一、项目目标

- 解决用户日常“吃什么”的选择困难症
- 提供个性化智能推荐
- 支持用户食材管理、口味偏好、忌口等
- 支持多端覆盖（网页、小程序、App）
- 提供完善的后台数据管理

---

## 二、用户角色

| 角色 | 描述 |
|:---|:---|
| 普通用户 | 浏览菜品、随机推荐、收藏、管理个人偏好和食材 |
| 管理员 | 登录后台，管理菜品、分类、食材、用户，查看数据统计 |

---

## 三、用户端页面（用户功能）

### 1. 首页 / 随机推荐页
- 随机展示一道菜（菜名、图片、菜系标签、口味标签、卡路里、价格、难度、时长）
- “换一个”按钮，随机刷新
- 快速筛选栏：按餐时（早/中/晚/宵夜/下午茶）切换推荐池
- 今日智能推荐（基于用户偏好 + 当前时间）
- 搜索入口（菜名、食材搜索）

### 2. 菜品详情页
- 菜品大图、名称、菜系、口味标签
- 营养信息：卡路里（可扩展蛋白质/碳水/脂肪）
- 参考价格、烹饪难度、烹饪时长
- 食材清单及用量
- 致敏物高亮提示
- 烹饪教程（图文步骤）
- 收藏/取消收藏按钮
- 与用户食材库对比，标记缺少的食材

### 3. 分类浏览页
- 按餐时浏览：早餐、午餐、晚餐、宵夜、下午茶
- 按菜系浏览：川菜、粤菜、湘菜、西餐、日料等
- 按口味浏览：辣、酸、甜、清淡、麻辣等
- 按场景浏览：一人食、家庭餐、老人餐、儿童餐
- 多维筛选：价格区间、卡路里区间、烹饪时长、难度

### 4. 智能推荐问卷（偏好设置）
- 用餐人数：单人 / 双人 / 全家（有老人 / 有小孩）
- 口味偏好：多选标签
- 忌口/过敏：花生、牛奶、海鲜、麸质等
- 偏好菜系：川菜、粤菜、日料等
- 倾心食材：鸡肉、豆腐、鸡蛋等
- 预算范围：经济实惠 / 正常 / 不计成本
- 卡路里目标：减脂 / 保持 / 增肌 / 不关心

### 5. 用户食材管理页
- 我的食材库：添加、删除、搜索家中现有食材
- 食材分类：蔬菜、肉类、调料、主食等
- “能做啥”：根据已有食材反向匹配可做的菜品
- 缺食材提醒：针对收藏的菜品标记缺少的食材

### 6. 收藏与历史页
- 我的收藏列表，支持取消收藏
- 推荐历史记录
- 浏览历史（可选）

### 7. 个人中心 / 设置页
- 修改个人资料（口味、忌口、菜系、预算、卡路里目标）
- 用餐人数设置
- 账号安全（修改密码、绑定手机/邮箱）
- 关于 / 反馈

### 8. 用户统计看板（个人版）
- 本周推荐最多的菜系（饼图）
- 收藏菜品口味分布（柱状图）
- 累计节省的“纠结时间”（趣味统计）
- 卡路里摄入趋势（折线图）


### 9. 用户反馈
- 投诉建议
- bug反馈


---

## 四、管理端页面（管理员功能）

### 1. 管理登录
- 管理员专用登录入口（或通过角色区分）

### 2. 仪表盘 / 数据总览
- 总菜品数、总用户数、今日推荐次数、总收藏数
- 热门菜品 Top 10
- 用户口味偏好分布（饼图）
- 各餐时推荐占比（柱状图）
- 新增用户趋势（折线图）

### 3. 菜品管理
- 菜品列表（分页、搜索、筛选）
- 新增/编辑菜品（全部属性字段）
- 批量导入/导出（可选）
- 删除菜品

### 4. 分类管理
- 餐时分类增删改查（早/中/晚/宵夜/下午茶）
- 菜系分类增删改查
- 口味标签管理

### 5. 用户管理
- 用户列表、搜索
- 查看用户画像（偏好、忌口、收藏数）
- 禁用/启用账号

### 6. 食材库管理
- 食材列表、分类
- 添加/编辑/删除食材

---

## 五、扩展功能（可选加分项）

- 分享卡片：生成菜品图片+二维码分享朋友圈
- 购物清单：根据选中的菜品自动生成购买清单
- 语音输入：Web Speech API 语音识别口味
- PWA 离线缓存
- 黑暗模式
- 多语言（vue-i18n）

## 六、技术实现规划

| 阶段 | 主要页面 | 核心技术 |
|:---|:---|:---|
| 一、Vue3 SPA | 首页推荐、菜品详情、分类浏览、收藏页、设置 | Vue3 Composition API、Pinia、Vue Router、TS |
| 二、Express 后端 | 用户注册登录、菜品API、收藏API、偏好推荐 | Express、JWT、Prisma、SQLite |
| 三、Nuxt + uni-app | 管理后台、食材管理、统计看板、小程序端 | Nuxt SSR、uni-app、Vercel部署 |
